### Resources : 
- Instance of services that you create, like virtual machines, storage, of SQL databases.

### Resources groups : 
- Resources are combined into resource groups.
- Logical container into which Azure resources like web apps, databases and storage accounts are deployed and managed.

### Subscriptions : 
- Groups together user accounts and the resources that have been created by those user accounts.
- For each subscriptions, there are limits or quotas on the amount of resources that are created by users, teams, or projects.

### Management groups : 
- Help manage access, policy, and compliance for multiple subscriptions.
- All subscriptions in a management group automatically inherit the conditions applied to the management group.
